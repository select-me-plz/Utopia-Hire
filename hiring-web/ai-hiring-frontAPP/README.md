# AI Hiring Frontend

This is the frontend application for the Hiring Web Application. It is built with Angular and provides a user-friendly interface for the hiring process.

## Features
- User authentication
- Dashboard for managing CVs
- Responsive design

## Installation
1. Navigate to the `ai-hiring-frontAPP` directory:
   ```bash
   cd ai-hiring-frontAPP
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Scripts
- Start the development server:
  ```bash
  npm start
  ```
- Build the application:
  ```bash
  npm run build
  ```

## Development
This project uses Angular CLI for development. To serve the application locally:
```bash
ng serve
```

## Dependencies
- Angular
- RxJS
- Zone.js
